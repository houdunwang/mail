# 邮件

邮件组件内部集成了方便的邮件发送机制，只需要修改相应配置项即可发送邮件，使用非常简单。

请 [查看在线手册](http://www.kancloud.cn/houdunwang/hdphp3/215233) 进行学习

[TOC]
#开始使用

####安装组件
使用 composer 命令进行安装或下载源代码使用。

```
composer require houdunwang/mail
```

##配置

```
$config = [
          	/*
          	|--------------------------------------------------------------------------
          	| 服务器使用ssl
          	|--------------------------------------------------------------------------
          	*/
          	'ssl'      => false,
          
          	/*
          	|--------------------------------------------------------------------------
          	| 帐号
          	|--------------------------------------------------------------------------
          	| 发送邮件的帐号
          	*/
          	'username' => 'dev@houdunwang.com',
          
          	/*
          	|--------------------------------------------------------------------------
          	| 密码
          	|--------------------------------------------------------------------------
          	| 发送邮件使用帐号的密码
          	*/
          	'password' => '',
          
          	/*
          	|--------------------------------------------------------------------------
          	| 主机
          	|--------------------------------------------------------------------------
          	| 邮件服务器的主机地址
          	*/
          	'host'     => 'smtp.houdunwang.com',
          
          	/*
          	|--------------------------------------------------------------------------
          	| 服务器端口号
          	|--------------------------------------------------------------------------
          	*/
          	'port'     => '25',
          
          	/*
          	|--------------------------------------------------------------------------
          	| 发件人名称
          	|--------------------------------------------------------------------------
          	| 收件人在收取邮件时显示的发件人名称
          	*/
          	'fromname' => '后盾网',
          
          	/*
          	|--------------------------------------------------------------------------
          	| 发件人邮箱地址
          	|--------------------------------------------------------------------------
          	| 收件人在收取邮件时显示的邮箱地址
          	| 与上面的帐号设置一样即可
          	*/
          	'frommail' => 'dev@houdunwang.com',
];
Config::batch($config);
```

##发送邮件

####函数语法
```
Mail::send(收件人邮箱,收件人名称,邮件标题,邮件内容);
```

####发送邮件
```
Mail::send(
    'xxx@sina.com',
    '后盾人',
    '邮件标题',
    '<h1 style="color:#f00;">邮件正文</h1>'
);
```
